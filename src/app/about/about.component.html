CountdownComponent

A Countdown timer

Component Decorator
Defines metadata, selector, imports, template, style files, and animations.
Constructor
Initializes the component.
ngOnInit
  • Loads title and date from local storage
  • Updates the countdown
  • Adjusts title font size
  • Sets interval to update countdown every second
ngAfterViewInit
Adjusts font size of the title and date elements after view initialization.
ngOnDestroy
Clears the interval if it exists when the component is destroyed.
loadFromLocalStorage
Loads event title and date from local storage and adjusts the title font size.
saveToLocalStorage
Saves event title and date to local storage.
updateTitle
Updates event title from input, saves to local storage, and adjusts font size.
adjustFontSize
Adjusts the font size of the title element based on its content.
adjustDateFontSize
Adjusts the font size of the date element based on its content.
onDateChange
Adjusts date font size, saves new date to local storage, and updates countdown.
updateCountdown
Updates countdown based on event date and current time, calculating days, hours, minutes, and seconds remaining.
